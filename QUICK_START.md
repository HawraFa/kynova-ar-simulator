# KyNova AR-Simulator - Quick Start (5 Minutes)

## 🎯 Fastest Deployment: GitHub Pages

### ⏱️ Total Time: 5 Minutes | Cost: $0

---

## Step 1: Customize (2 minutes)

Open `index.html` and replace:

1. **Find & Replace** (Ctrl+H):
   ```
   [Your University Name]     →  Your actual university
   [Team Member 1]           →  First member's name
   [Team Member 2]           →  Second member's name
   [Team Member 3]           →  Third member's name
   [Team Member 4]           →  Fourth member's name
   [Student ID]              →  Actual student IDs
   [Supervisor Name]         →  Your supervisor's name
   [Department]              →  Your department
   ```

2. **Update email** (line ~520):
   ```html
   <p>info@kynova-ar.com</p>  →  <p>your-email@example.com</p>
   ```

---

## Step 2: Deploy to GitHub (3 minutes)

### A. Create Repository
1. Go to https://github.com/new
2. Name: `kynova-ar-simulator`
3. Public
4. Create

### B. Upload Files
1. Click "uploading an existing file"
2. Drag these files from `website/` folder:
   - `index.html`
   - `css/style.css` (keep folder structure)
   - `js/main.js` (keep folder structure)
3. Commit

### C. Enable Pages
1. Settings → Pages
2. Source: main branch
3. Save

### D. Get Your URL
Wait 1 minute, then visit:
```
https://YOUR-USERNAME.github.io/kynova-ar-simulator
```

## 🆘 Problems?

**Website not loading?**
- Wait 2 minutes
- Clear browser cache (Ctrl+Shift+R)

**CSS not showing?**
- Check folder structure is correct: `css/style.css`, `js/main.js`
