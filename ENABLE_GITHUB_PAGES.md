## 🆘 TROUBLESHOOTING

**Issue: GitHub Pages not showing up in Settings**
- Solution: Make sure repository is public (Settings → General → Danger Zone → Change visibility)

**Issue: Website not loading after enabling Pages**
- Solution: Wait 2-5 minutes, then clear browser cache (Ctrl+Shift+R)

**Issue: Website loads but CSS/styling missing**
- Solution: This shouldn't happen with current setup, but verify file paths are correct

**Issue: 404 Error**
- Solution: Ensure `index.html` is in root directory of repository (not in subfolder)
